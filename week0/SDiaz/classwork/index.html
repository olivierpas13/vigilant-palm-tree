Internet & IPS

Internet: Es una red global de dispositivos conectados mediante protocolos, el Protocolo de Internet (IP) es uno de los protocolos más importantes en esta conexión.

IPS (Internet Protocol Suite): es un conjunto de protocolos de comunicación utilizados para la transmisión de datos a través de redes, contiene 4 capas importantes:

1 Aplicación

Son servicios de red que permiten que el sw interactúe con la red, usa protocolos como HTTP, HTTPS.

2 Transporte

Sirve para gestionar la transmisión de datos mediante el protocolo TCP (Transmission Control Protocol).

Los mensajes SYN, SYN-ACK, ACK son utilizados durante el proceso de establecer conexión entre 2 dispositivos.

Cliente --SYN-> Servidor: se pide una conexión.

Servidor --SYN-ACK-> Cliente: se confirma la solicitud y se sincroniza.

Cliente --ACK-> Servidor: se confirma la respuesta del servidor.

3 Network

Es la dirección o la ruta de los paquetes a traves de la red, se usan direcciones IP (Internet Protocol).

  • IPv4: Protocolo de 32 bits, puede llegar máx hasta 255, 4 números separados por puntos.
  • IPv6: Protocolo de 128 bits, usa hexadecimales y está separado por 2 puntos (:).

4 Datalink

Tiene que ver con el hardware, es decir, conexión física sus protocolos se dan mediante wi-fi, etc.

WWW & HTTP

La WWW (World Wide Web): es una colección global de recursos en la red.

HTTP y HTTPS: son protocolos que se utilizan para transferir recursos desde los servidores hasta el navegador.

HTTP --> comunicación básica, HTTPS --> agrega seguridad cifrando la información transferida.

Conexión

  • URL (Uniform Resource Locator): es la dirección web que se escribe para acceder a una ubicación (página web).
  • BROWSER: conocido como cliente, es el programa que se usa para acceder a las páginas web (como Chrome, Firefox, etc).
  • SERVER es el que recibe la solicitud, busca el recurso y envía una respuesta al navegador.
  • DOMAIN: es la parte de la URL que identifica el nombre de un sitio web.
  • DNS (Domain Name System): es el que traduce los nombres de los dominios a una dirección IP.

Estructura de una página Web

  • HTML (HyperText Markup Language): es el lenguaje de marcado utilizado para estructurar el contenido en la web, es decir los huesos o el esqueleto.
  • JavaScript: es un lenguaje de programación que permite añadir interactividad a una página web, se podría decir que equivale a los músculos.
  • CSS: es el lenguaje con el que le damos estilos al contenido etiquetado con HTML, se asemeja a la piel del ser humano.
