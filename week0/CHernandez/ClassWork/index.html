Internet

EL internet es una red de dispositivos conectados, los mismos proporcionan una comunidad de informacion a la cual se puede acceder usando direcciones ip, si bien el termino internet es usado para referirse a la parte del software, este hace mencion a la infraestructura fisica que lo compone. es decir el cableado, los servidores fisicos y los dispositivos que proveen acceso a la red.

Protocols

Data Link

Es el nivel mas bajo de los protocolos, engloba a la parte fisica de la conexion a la red, el protocolo en si es el preparar el hardware para que no se pierda informacion durante el camino a entregarla, en si es las condiciones que se tiene en el cableado, servidores y routers

Network

Es el segundo nivel de los protocolos, es la direccion y la identificacion unica de cada dispositivo con acceso a la web, dentro de la ip existen dos versiones que son las mas usadas, siendo el IPV4 y el IPV6 las mas usadas, la unica diferencia entre ambas es la cantidad de bits que usan para detallar la direccion de dicho dispositivo.
El IP es la direccion mediante la web reconoce donde se encuentran las cosas, es decir: servidores web, aplicaciones o paginas.

Transport

En este protocolo tenemos el TCP(Transmission Control Protocol), este protocolo es el primer filtro en la parte relacionada al software, es la que se encarga de crear un puente entre el cliente y el servidor es decir manda un box de prueba que va como un explorador para observar si existe una conexion.
Tambien se encarga de crear un cuello de botella que fragmenta la informacion en chunks mas peque√±os ayudando a evitar que la red o puente se sature. Este protocolo es independiente, quiere decir que debes hacerlo cada vez que busques establecer una conexion con un servidor o pagina web

Aplicaciones

Aqui tenemos los protocolos usados para paginas web o Aplicaciones que tiene su ubicacion en la red, el mas usado es Http el cual se basa en reglas de creacion y de verificacion de sitios web, en este se usa reglas de sintaxis que deben seguir todas las paginas, siguen el modelo cliente-servidor, aqui se usa html semantico, su ultima evolucion fue a HTTPS, en donde la s es de seguro, pues se encuentra increptado.

Structures

Se usan tres partes fundamentales para crear algo en la web, estas son:

  • HTML

    Hypertext Markup Language is the standard markup language used to create and structure content on the web. Es el que da la estructura

  • CSS

    Cascading Style Sheets, is a stylesheet language used to define the visual appearance and formatting of a web page, es el que da la apariencia

  • JavaScript

    High-level, dynamic , multiparadigm programming - language primarily used for adding interactivity, functionality, and dynamic behavior to web pages, es el que da el backend y funciones
